# node_console_ask
A Node.js module for asking questions and receiving an array of answers via a Promise object.

The module incorporates Node's `readline.createInterface()` to coordinate a sequence of console based questions and recording their respective answers.

The file `index.js` provides a demo using the module.  In a command prompt, go to the module's directory and enter `node index.js`.

Node.js version 16.8.0 was used in support of this module.
